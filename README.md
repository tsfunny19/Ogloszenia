# OgloszeniaSCO

Nowoczesny, lekki i w pełni konfigurowalny plugin do automatycznych ogłoszeń dla serwerów **Minecraft Paper 1.21.x**.

## ✨ Funkcje

- 📢 Automatyczne ogłoszenia na czacie
- ⚡ Wysoka wydajność i optymalizacja
- ⚙️ Pełna konfiguracja w `config.yml`
- 📝 Łatwa edycja ogłoszeń
- 🎲 Losowa lub sekwencyjna kolejność wiadomości
- 🎨 Obsługa kolorów MiniMessage
- 🌈 Obsługa kolorów RGB i gradientów
- 🔌 Integracja z PlaceholderAPI
- 📌 Konfigurowalny prefiks wiadomości
- ⏱ Możliwość ustawienia interwału wysyłania
- 🔄 Przeładowanie konfiguracji bez restartu serwera
- ☕ Java 21
- 📦 Projekt oparty na Maven
- 🧩 Wsparcie dla Paper 1.21.x

## 📥 Instalacja

1. Pobierz najnowszą wersję pluginu.
2. Umieść plik `.jar` w folderze `plugins` swojego serwera.
3. Uruchom lub zrestartuj serwer.
4. Skonfiguruj plugin w pliku `config.yml`.
5. Gotowe!
